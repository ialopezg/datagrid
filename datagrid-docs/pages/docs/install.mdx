## Installation

1. Install Peer Dependencies

> `@ialopezg/datagrid` requires **Material UI V5** and **React Table V7** as Peer Dependencies.If you are already using
Material UI, you probably already have most of these peer dependencies installed.
>
> If you are already using Material UI, you probably already have most of these peer dependencies installed.
>
> Just double check that you have the following in your package.json, or just use the [full install commands](<#install-with-required-peer-dependencies-(recommended)>) below.
>
> 1.  react-table (v7)
> 2.  @mui/material (v5)
> 3.  @mui/icons-material (v5)
> 4.  @emotion/react (v11)
> 5.  @emotion/styled (v11)

### Quick Install

#### NPM
```bash
npm install @ialopezg/datagrid
```

#### PNPM

```bash
pnpm add @ialopezg/datagrid
```

#### Yarn

```bash
yarn add @ialopezg/datagrid
```

### Install With Required Peer Dependencies (Recommended)

#### NPM

```bash
npm install @ialopezg/datagrid react-table @mui/material @mui/icons-material @emotion/react @emotion/styled
```

#### PNPM

```bash
pnpm add @ialopezg/datagrid react-table @mui/material @mui/icons-material @emotion/react @emotion/styled
```

#### Yarn

```bash
yarn add @ialopezg/datagrid react-table @mui/material @mui/icons-material @emotion/react @emotion/styled
```
